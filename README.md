Welcome to my interactive 3D portfolio! This project is a lovingly crafted, witchcraft-themed digital sanctuary built with **Three.js** and **React Three Fiber**. It is designed to be more than just a showcase of my work—it's a little interactive haven where you can wander a mystical island, sit by a crackling campfire, and explore my creations at your own pace. 

> ⚠️ **Note:** This forest is currently a **Work in Progress**! I am continually planting new code and placing magic stones. Please pardon any stray pixie dust or broken twigs as you explore. 🌱🚧

## ✨ Magical Features

- **Cozy 3D Environment:** Wander around a stylized woodland island filled with giant mushrooms, wispy grass, and hand-placed stones.
- **Witchy Atmosphere:** Glowing flora, floating magical particles, and a warming campfire set a tranquil, mystical mood.
- **Dynamic Day & Night:** Watch the digital sky transition as day turns to night and the stars come out.
- **Immersive Ambience:** Soothing audio, immersive exploration, and beautiful post-processing water effects.

## 📜 Brewing Instructions (Local Setup)

Want to explore the forest on your local machine? 

1. **Install the ingredients:**
   ```bash
   npm install
   ```
2. **Ignite the fire (Start the dev server):**
   ```bash
   npm run dev
   ```
3. **Enter the portal:** 
   Open `http://localhost:5173` in your favorite browser.

## 🌲 Realm Structure

- `src/scene/` - The core architecture behind the 3D world (Campfire, Island, Water, DayNight cycle).
- `src/ui/` - The stylized HUD and overlays that gently guide your journey.
- `src/audio/` - Hooks that brew the background ambient sounds.
- `public/models/nature/` - The 3D assets, trees, and flora making up our witchy woods.
